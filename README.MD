# **Engenharia de Software III**

Repositório criado para apoiar as aulas da disciplina de **Engenharia de Software III** do curso de Bacharelado em Sistemas de Informação do IF Sudeste MG - Campus Manhuaçu.

- **Objetivo principal**: capacitar os alunos em práticas ágeis de desenvolvimento de software, bem como na adoção de técnicas e teconologias de implementação do **zero ao deploy** em sistemas de informação baseados na web. 
- **Objetivos específicos**:
  - Desenvolver sistema de e-commerce (simples) como mecanismo didático para apoiar o aprendizado basedo em projetos e *learning by doing*
  - Utilizar as principais tecnologias full stack do mercado, tais como Node.js, Express.js, React, MongoDB, Jest Git, GitHub, Render, Vercel, Docker, dentre outros.
  - Resolver problemas de engenharia de software baseados em fontes técnicas (ex: documentações) e científicas (artigos científicos)
  - Gerenciar projetos de software
  - Planejar e executar experimentos envolvendo projetos de software
