# **Engenharia de Software III**

Repositório criado para apoiar as aulas da disciplina de **Engenharia de Software III** do curso de Bacharelado em Sistemas de Informação do IF Sudeste MG - Campus Manhuaçu.

- **Objetivo principal**: capacitar os alunos em práticas ágeis de desenvolvimento de software, bem como na adoção de técnicas e tecnologias de implementação do **zero ao deploy** em sistemas de informação baseados na web, incorporando o uso **consciente e crítico de LLMs (Large Language Models)** no apoio às atividades de Engenharia de Software.

- **Objetivos específicos**:
  - Desenvolver sistema de e-commerce (simples) como mecanismo didático para apoiar o aprendizado baseado em projetos e _learning by doing_
  - Utilizar as principais tecnologias full stack do mercado, tais como Node.js, Express.js, React, MongoDB, Jest, Git, GitHub, Render, Vercel, Docker, dentre outros
  - Integrar o uso de LLMs em atividades de Engenharia de Software (ex: geração de código, testes, documentação e refatoração), de forma **orientada, crítica e baseada em evidências**
  - Avaliar a qualidade das soluções geradas por LLMs, considerando aspectos como corretude, manutenibilidade e aderência a boas práticas
  - Resolver problemas de engenharia de software com base em fontes técnicas (documentações) e científicas (artigos), utilizando LLMs como ferramenta de apoio, e não como substituto do raciocínio técnico
  - Gerenciar projetos de software com suporte de ferramentas modernas, incluindo integração com assistentes baseados em IA
  - Planejar e executar experimentos envolvendo projetos de software, incluindo a análise do impacto do uso de LLMs no processo de desenvolvimento
